# Trader Performance vs Market Sentiment

This project analyzes the relationship between trader behavior on Hyperliquid and Bitcoin market sentiment (Fear & Greed Index).

## Contents
- notebook_1.ipynb: Main analysis notebook (Google Colab)
- csv_files/: Raw and processed datasets
- outputs/: Charts generated during analysis
- ds_report.pdf: Summary of insights and strategies

## How to Run
1. Open notebook_1.ipynb in Google Colab
2. Upload CSV files from csv_files/
3. Run all cells top to bottom
